#  Cloud Posture Dashboard

##  Problem Statement
Cloud security teams managing multiple cloud accounts (AWS, Azure, GCP) face difficulty in identifying and tracking misconfigurations due to lack of centralized visibility. This leads to delayed issue detection and increased security risks.

---

##  User Persona
**Role:** Cloud Security Engineer  
Responsible for monitoring and securing cloud environments across multiple platforms.

---

##  Solution
A centralized dashboard that allows users to:
- View all cloud misconfigurations in one place  
- Filter issues by cloud, region, and severity  
- Quickly investigate and fix issues  

---

##  Key Features

### 1. Dashboard Overview
- Total issues count  
- Severity breakdown (Critical, Medium, Low)  

### 2. Filters
- Cloud (AWS, Azure, GCP)  
- Region  
- Severity  

### 3. Issue List
- Shows all detected issues  
- Includes severity, cloud provider, and status  

### 4. Issue Detail View
- Detailed description  
- Affected account and region  
- Recommended fix  

### 5. Fix Action
- “Fix Now” button for quick resolution  

---

##  Feature Prioritization
1. Dashboard overview  
2. Filtering system  
3. Issue detail view  
4. Fix action  

---

##  Success Metrics
- Reduced issue detection time  
- Reduced issue resolution time  
- Fewer critical issues  

---
